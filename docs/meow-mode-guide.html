🇰🇷 한국어

1. 설치

  1. 내 맥에 맞는 .dmg를 받으세요.
    • 애플 실리콘(M1/M2/M3/M4): Meow Mode-x.x.x-arm64.dmg
    • 인텔: Meow Mode-x.x.x.dmg
    • 모르겠으면 애플 메뉴 → 이 Mac에 관하여에서 칩 확인.
  2. .dmg를 열고 Meow Mode를 응용 프로그램 폴더로 끌어다 놓기.

2. 첫 실행 (중요!)

서명/공증을 하지 않은 앱이라, 처음엔 macOS가 막습니다. 안전하니 아래대로 한 번만 풀어주세요.
  • 방법: 응용 프로그램에서 Meow Mode를 우클릭 → 열기 → (다시) 열기.
  • macOS Sequoia(15) 이상: 더블클릭 → 차단되면 → 시스템 설정 → 개인정보 보호 및 보안 → 아래쪽 "Meow Mode이(가) 차단되었습니다" 옆 "그래도 열기".
한 번만 열어주면, 그 다음부터는 그냥 더블클릭으로 열립니다.

3. 기본 사용법

  • 🐾 고양이는 알아서 돌아다녀요. 드래그해서 옮길 수 있어요.
  • 🏠 집도 드래그로 이동. 지붕의 ⚙️를 누르면 설정이 열려요.
  • 🍚 밥그릇·물그릇을 클릭하면 채워집니다. 배고프면 고양이가 보채요.
  • 🖱️ 마우스를 고양이 가까이 가져가면 쳐다봐요.

4. 메뉴바 아이콘 (화면 맨 위 고양이 얼굴)

잠깐 끄기 / 다시 켜기고양이를 잠시 숨겼다 다시 부르기 (아이콘이 자는 얼굴로 바뀜)
고양이 찾기어디 있는지 모를 때 — 화면 가운데로 잠깐 불러옴
화면 위치맨앞 / 중간 / 맨뒤 — 다른 창과의 순서 조절
종료앱 끄기

5. 설정 (지붕의 ⚙️)

  • 이름 바꾸기, 활동량(조용/보통/활발), 활동 영역
  • 소리: 끔 / 기본 / 아기 / 굵은 — 누르면 미리 들려요
  • 말풍선 켜기·끄기, 언어(한/영/일/중)

6. 문제 해결

"손상되어 휴지통으로" 뜸위 2번대로 우클릭→열기 (또는 시스템 설정에서 "그래도 열기").
아이콘이 안 바뀜macOS 아이콘 캐시 탓. 로그아웃 후 다시 로그인하면 갱신됩니다.
고양이가 안 보임메뉴바 아이콘 → 고양이 찾기. 또는 잠깐 끄기 상태인지 확인.
클릭이 고양이에 막힘고양이/집/그릇 위가 아니면 클릭은 그대로 통과합니다. 정상이에요.

🇬🇧 English

1. Install

  1. Download the right .dmg for your Mac:
    • Apple Silicon (M1/M2/M3/M4): Meow Mode-x.x.x-arm64.dmg
    • Intel: Meow Mode-x.x.x.dmg
    • Not sure? Apple menu → About This Mac → check the chip.
  2. Open the .dmg and drag Meow Mode into your Applications folder.

2. First launch (important!)

The app isn't notarized, so macOS blocks it the first time. It's safe — just allow it once below.
  • How: right-click Meow Mode in Applications → Open → Open again.
  • macOS Sequoia (15)+: double-click → if blocked → System Settings → Privacy & Security → near the bottom, next to "Meow Mode was blocked", click "Open Anyway".
After allowing it once, it opens with a normal double-click from then on.

3. Basics

  • 🐾 The cat wanders on its own. Drag it to move it.
  • 🏠 Drag the house too. Click the ⚙️ on the roof for Settings.
  • 🍚 Click the food / water bowls to refill them. It nags when hungry.
  • 🖱️ Move your cursor near the cat and it looks at it.

4. Menu bar icon (cat face, top of screen)

Pause / ResumeHide the cat for a bit, then bring it back (icon shows sleepy eyes)
Find the catLost it? Briefly calls it to the center of the screen
Window layerFront / Middle / Back — how it stacks against other windows
QuitClose the app

5. Settings (⚙️ on the roof)

  • Rename the cat, activity level, roam area
  • Sound: Off / Classic / Kitten / Deep — tap to preview
  • Speech bubbles on/off, language (KO/EN/JA/ZH)

6. Troubleshooting

"…is damaged, move to Trash"Do step 2: right-click → Open (or "Open Anyway" in System Settings).
Icon didn't updateIt's the macOS icon cache. Log out and back in to refresh.
Can't see the catMenu bar icon → Find the cat, or check if it's paused.
Clicks blocked by the catClicks pass through everywhere except on the cat/house/bowls. That's normal.

🇯🇵 日本語

1. インストール

  1. お使いの Mac に合った .dmg をダウンロード：
    • Apple Silicon（M1/M2/M3/M4）: Meow Mode-x.x.x-arm64.dmg
    • Intel: Meow Mode-x.x.x.dmg
    • 不明な場合は Apple メニュー → この Mac について でチップを確認。
  2. .dmg を開き、Meow Mode を アプリケーション フォルダへドラッグ。

2. 初回起動（重要！）

公証していないアプリのため、初回は macOS がブロックします。安全なので、下記の手順で一度だけ許可してください。
  • 方法: アプリケーションの Meow Mode を 右クリック → 開く → もう一度「開く」。
  • macOS Sequoia(15) 以降: ダブルクリック → ブロックされたら → システム設定 → プライバシーとセキュリティ → 下の方「Meow Mode はブロックされました」の横の 「このまま開く」。
一度許可すれば、次回からは普通のダブルクリックで開きます。

3. 基本の使い方

  • 🐾 猫は自由に歩き回ります。ドラッグで移動できます。
  • 🏠 おうちもドラッグで移動。屋根の ⚙️ で 設定が開きます。
  • 🍚 ごはん・水のお皿をクリックで補充。お腹がすくと催促します。
  • 🖱️ カーソルを近づけると、猫が見つめます。

4. メニューバーのアイコン（画面上部の猫の顔）

一時停止 / 再開猫を少し隠して、また呼び戻す（アイコンが寝顔に変わります）
猫を探す見失ったとき、画面中央に少し呼び寄せます
表示レイヤー最前 / ふつう / 最背 — 他のウインドウとの重なり順
終了アプリを終了

5. 設定（屋根の ⚙️）

  • 名前の変更、活動量、行動エリア
  • 鳴き声: オフ / 標準 / 子猫 / 太い — タップで試聴
  • 吹き出しのオン・オフ、言語（韓/英/日/中）

6. トラブルシューティング

「壊れている、ゴミ箱へ」手順 2：右クリック → 開く（または設定で「このまま開く」）。
アイコンが変わらないmacOS のアイコンキャッシュです。ログアウトして再ログインで更新。
猫が見えないメニューバー → 猫を探す、または一時停止中か確認。
クリックが猫にさえぎられる猫・家・お皿の上以外はクリックがそのまま通ります。正常です。

🇨🇳 中文

1. 安装

  1. 下载适合你 Mac 的 .dmg：
    • Apple 芯片（M1/M2/M3/M4）: Meow Mode-x.x.x-arm64.dmg
    • Intel: Meow Mode-x.x.x.dmg
    • 不确定？点击苹果菜单 → 关于本机 查看芯片。
  2. 打开 .dmg，将 Meow Mode 拖入 应用程序 文件夹。

2. 首次启动（重要！）

应用未经公证，首次会被 macOS 拦截。它是安全的 —— 按下面操作允许一次即可。
  • 方法: 在应用程序里 右键点击 Meow Mode → 打开 → 再次「打开」。
  • macOS Sequoia(15) 及以上: 双击 → 被拦截 → 系统设置 → 隐私与安全性 → 底部「Meow Mode 已被阻止」旁点 "仍要打开"。
允许一次后，之后双击即可正常打开。

3. 基本用法

  • 🐾 猫咪会自己活动。拖动可移动它。
  • 🏠 小屋也能拖动。点击屋顶的 ⚙️ 打开设置。
  • 🍚 点击食碗 / 水碗补充。饿了它会催你。
  • 🖱️ 把光标靠近猫咪，它会盯着看。

4. 菜单栏图标（屏幕顶部的猫脸）

暂停 / 继续暂时藏起猫咪再唤回（图标变成睡眼）
寻找猫咪找不到时，把它短暂唤到屏幕中央
窗口层级最前 / 中间 / 最后 —— 与其他窗口的层叠顺序
退出关闭应用

5. 设置（屋顶的 ⚙️）

  • 重命名、活跃度、活动范围
  • 叫声：关 / 默认 / 幼猫 / 低沉 —— 点击试听
  • 对话气泡开关、语言（韩/英/日/中）

6. 疑难解答

“已损坏，移到废纸篓”按第 2 步：右键 → 打开（或在系统设置点“仍要打开”）。
图标没更新是 macOS 图标缓存。注销后重新登录即可刷新。
看不到猫咪菜单栏图标 → 寻找猫咪，或检查是否处于暂停。
点击被猫咪挡住除猫/屋/碗之外，点击都会正常穿透。这是正常的。
