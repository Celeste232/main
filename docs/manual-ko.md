> 데스크탑에 살짝 못 그린 고양이가 한 마리 들어옵니다. 집도 있고 밥도 줘야 해요.

발그림 그대로의 컨셉을 좋아하지 않으셨다면 환불해드립니다. (안 합니다)

## 1. Meow Mode가 뭐예요

데스크탑 위에 떠다니는 작은 고양이를 키우는 앱입니다. 다른 일 하다가 가끔 보면 식빵을 굽고 있거나 자고 있거나 갑자기 "심심해" 같은 헛소리를 합니다.

**구성**
- 발그림 고양이 한 마리
- 집 한 채 (위치 옮길 수 있음)
- 밥그릇, 물그릇 (비우면 보채요)
- 메뉴바 트레이 아이콘

**작동 환경**
- macOS 12 (Monterey) 이상
- Apple Silicon (M1/M2/M3/M4) 또는 Intel
- 약 200 MB 디스크 공간

## 2. 설치하기

### 2-1. 다운로드

- **Apple Silicon Mac** (M 시리즈) → `MeowMode-x.x.x-arm64.dmg`
- **Intel Mac** → `MeowMode-x.x.x.dmg`

본인 Mac이 어느 쪽인지 모르겠다면 좌상단 사과 → "이 Mac에 관하여" → "칩" 항목 확인.

### 2-2. 설치

1. 받은 `.dmg` 파일을 더블클릭
2. 작은 창이 열리면서 MeowMode 아이콘과 **응용 프로그램** (Applications) 폴더가 보임
3. **MeowMode 아이콘을 응용 프로그램 폴더로 드래그**
4. 디스크 이미지 창은 닫고, 마운트된 디스크는 우클릭 → "꺼내기" (또는 그냥 둬도 됨)

## 3. 첫 실행 시 macOS 경고 우회하기

**⚠️ 처음 한 번만 필요합니다. 그 뒤로는 그냥 더블클릭하면 됩니다.**

이 앱은 **Apple Developer 사이닝이 되어있지 않습니다.** (소규모 개인 제작이라 연 $99 사이닝 비용을 안 들였어요.) 그래서 macOS가 첫 실행 시 "확인되지 않은 개발자" 경고를 띄웁니다. **악성 코드가 아니라, 그냥 Apple에 등록 안 한 개발자라는 뜻입니다.** 우회 방법:

### macOS Sonoma (14) 이하

1. 응용 프로그램 폴더에서 **MeowMode를 우클릭** (또는 Control + 클릭)
2. **"열기"** 선택
3. 경고 창에서 다시 **"열기"** 클릭

### macOS Sequoia (15) 이상

우클릭 → 열기 방식이 막혀있어요. 대신:

1. 그냥 더블클릭으로 한 번 시도 → 경고 뜨면 닫기
2. 좌상단 사과 → **시스템 설정** → **개인정보 보호 및 보안**
3. 맨 아래 "보안" 섹션에서 **"'MeowMode'은(는) 차단되었습니다"** 메시지 옆 **"그래도 열기"** 클릭
4. 비밀번호 입력 → 다시 한 번 "열기"

한 번만 하면 이후로는 그냥 더블클릭하면 됨.

## 4. 화면 구성

앱이 켜지면 화면 어딘가에 이렇게 나타납니다:

```
           🌑(톱니)
           ●
          🏠 (집)        🐱 (고양이)
          🐟 (생선현판)
          🍚 🥛 (밥/물그릇)
```

| 요소 | 설명 |
|---|---|
| **고양이** | 화면을 돌아다님. 자세도 바뀌고 말도 함 |
| **집** | 고양이가 들어가는 곳. 위치 옮길 수 있음 |
| **생선 현판** | 집 정문 위 작은 생선. 별 기능 없음 |
| **밥그릇** | 음식 양이 보임. 비면 채워줘야 함 |
| **물그릇** | 물 양이 보임 |
| **톱니** ⚙️ | 집 위에 떠있는 작은 동그라미. 클릭하면 설정 메뉴 |
| **메뉴바 트레이** | macOS 상단 메뉴바에 고양이 머리 아이콘 |

**투명 창**: 화면 전체를 덮는 투명한 창이지만, 고양이/집/그릇 위에서만 클릭이 먹히고 나머지 공간은 다른 앱이 그대로 사용됩니다. 마우스 통과해요.

## 5. 기본 조작

| 동작 | 결과 |
|---|---|
| 집을 클릭 | 고양이가 집으로 옵니다 ("갈게~") |
| 집을 드래그 | 집의 위치가 바뀜. 옆에 밥/물그릇도 따라옴 |
| 고양이가 잘 때 집 드래그 | 자는 고양이도 집과 함께 옮겨감 |
| 톱니 ⚙️ 클릭 | 설정 메뉴 열림 |
| 고양이를 드래그 | 들어 올림 ("내려놔!" / "Put me down!"). 놓으면 놀람 |
| 마우스를 고양이 근처에 | 고양이가 마우스를 쳐다봄 |
| 밥그릇 클릭 | 사료 가득 채워짐 |
| 물그릇 클릭 | 물 가득 채워짐 |

## 6. 메뉴바 트레이

상단 메뉴바의 작은 고양이 머리 아이콘을 통해 빠르게 조작할 수 있어요.

### 좌클릭 (한 번 누르기)

**잠깐 끄기 / 다시 켜기 토글.** 작업 집중하고 싶을 때 좌클릭하면 고양이가 숨음. 트레이 아이콘에 `z` 표시가 붙음. 다시 좌클릭하면 돌아옴.

### 우클릭 (메뉴 펼치기)

- **잠깐 끄기 / 다시 켜기** — 위와 동일
- **고양이 찾기** — 어디 있는지 모를 때. 현재 위치에서 폴짝폴짝 뛰면서 "여기야~!". 화면 뒤에 있으면 잠깐 앞으로 나옴
- **화면 위치**
  - **맨 앞** — 다른 창 위에 항상 보임 (기본)
  - **보통** — 일반 창처럼 동작
  - **맨 뒤** — 작업 중 방해 안 되게 뒤로 보냄
- **종료** — 앱 끄기

## 7. 설정 메뉴 상세

집 위 톱니 ⚙️ 클릭하면 손그림 스타일 설정 패널이 열립니다.

### 고양이 섹션

**이름**: 고양이 이름. 채터링에 사용됨. 기본 "나비". (영문: Nabi / 일문: Tama / 중문: 小猫)

**스킨**: 현재 "기본" (발그림). 추후 추가 스킨팩 구매 시 여기서 선택 가능.

**활동량**: 윷놀이판 슬라이더. 흰 생선이 현재 값을 가림.
- **조용** — 거의 자고 있음. 식빵, 퍼져있기 위주
- **보통** — 적당히 돌아다님 (기본)
- **활발** — 자주 뛰어다니고 마우스 쫓아다님

**활동영역**: 어디까지 돌아다닐지.
- **집근처** — 집에서 350px 안에서만
- **가장자리** — 화면 네 모서리 따라 돌아다님
- **전체** — 화면 전체 (기본)

> 집안에서만 있게 하려면 아래 **"집에 넣기"** 버튼을 누르세요.

**말풍선**: 고양이 말풍선 표시 여부.

**고양이 숨기기**: 고양이만 안 보이게 (집과 그릇은 그대로).

### 액션 버튼

- **고양이 찾기** — 트레이 메뉴와 동일
- **집으로 부르기** — 집까지 걸어옴
- **집에 넣기 / 집에서 꺼내기** — 집 안에 가둠. 머리만 문틈으로 보임. 다시 누르면 나옴

### 프로그램 섹션

**화면 위치**: 트레이 메뉴와 동일.

**잠깐 끄기**: 트레이 좌클릭과 동일.

**자동 실행**: macOS 로그인 시 자동으로 켜기.

**언어**: 한국어 / English / 日本 / 中文 즉시 전환.

### 데이터 섹션

**친밀도**: 고양이와 상호작용한 횟수. 밥/물 채워주면 올라감. (현재는 표시만)

**초기화**: 설정을 처음 상태로 (이름, 위치, 스킨 다 리셋).

**숨기기**: 창만 숨김. 트레이 아이콘에서 다시 켜야 함.

**종료**: 앱 완전히 끄기.

## 8. 고양이가 하는 행동

활동량/영역 설정과 상관없이 무작위로 이런 행동을 합니다:

**일상**
- 걷기, 앉기, 식빵 굽기 (납작 자세), 퍼져있기, 슈퍼맨 자세 (팔다리 쭉 뻗기)
- 자기, 그루밍 (세수), 스트레칭, 하품
- 꼬리 흔들기, 호기심 (고개 갸웃)

**감정 표현**
- 행복 (^_^ + 하트), 놀람 (!!), "잡혔다" (눈 동그래짐)

**갑작스러운 짓**
- 재채기 ("에취!")
- 덮치기 (점프 자세)
- 데굴데굴 구르기
- 부르르 털기
- 야옹 ("야옹", "심심해", "흥", "쳐다보지 마" 등)
- 별별 표정 ("✨" 반짝눈)

**드물게 발생 (화면 위치 "맨 앞" 모드일 때만)**
- 미끄러지기 ("어어!") — 보이지 않는 창 가장자리에서
- 매달리기 ("!?") — 점선으로 표시된 보이지 않는 선반에 앞발로 매달림
- 올라오기 ("으샤!") — 발톱자국 남기면서 다시 올라옴

**말풍선 채터링**

화면이 조용해지면 가끔 혼잣말:

- "{이름}랑 놀자!"
- "흥"
- "싫음 말고"
- "갈게"
- "야옹"
- "심심해"
- "낮잠 잘래"
- "쳐다보지 마"

**배고픔 / 목마름 알림**

밥/물이 거의 비면 "밥줘!" / "물 없어!" 가끔 외침.

## 9. 숨겨진 재미

- **자는 고양이는 집 옮길 때 같이 따라가요** — 깨우지 않으려고 평소엔 안 따라가는데, 잘 때만 따라감
- **마우스가 가까이 가면 쳐다봄** — 200px 안에 머무르면 고개를 그쪽으로 돌림. 80px 안에서 오래 있으면 가끔 호기심 자세
- **밥 먹고 나면 "맛있어!"** + 하트 표정
- **고양이를 들고 있다 놓으면 놀람** — "내려놔!" 외치고 startled 자세
- **창 위에서 미끄러질 때** 발톱자국이 살짝 남았다가 사라짐

## 10. 자주 묻는 질문

**Q. 영상통화/회의 때 화면공유에 고양이가 보여요!**
→ 메뉴바 트레이 좌클릭으로 잠깐 끄기. 또는 설정 → 화면 위치 → "맨 뒤".

**Q. 게임 화면에 가려요 / 풀스크린 앱에서 안 보여요**
→ 풀스크린 모드는 디자인상 무시됩니다. 사용자 작업을 방해하지 않으려고 일부러 그래요.

**Q. 다른 모니터에서 안 보여요**
→ 현재 버전은 **가장 왼쪽 모니터**에서만 동작합니다. 멀티 모니터 선택은 추후 업데이트 예정.

**Q. 메모리 / CPU 부담은 어떤가요?**
→ 평균 50~80MB RAM, CPU 1% 미만. 노트북 발열 거의 없음.

**Q. 자동으로 켜지게 하려면?**
→ 설정 → 자동 실행 → 켬.

**Q. 다 짜증 났는데 어떻게 끄나요?**
→ 메뉴바 트레이 우클릭 → 종료. 또는 설정 → 종료.

## 11. 문제 해결

**고양이가 안 보여요**
1. 설정 → 고양이 숨기기 → 보임
2. 트레이 좌클릭으로 "잠깐 끄기" 해제
3. 다른 모니터 사용 중이면 메인 모니터로 옮겨주세요

**클릭이 안 먹혀요**
- 고양이/집/그릇 정확히 클릭해야 함. 빈 공간은 다른 앱이 받음
- 만약 톱니가 안 눌리면 마우스를 톱니 위로 잠깐 가져갔다가 클릭

**설치 후 아이콘이 옛날 거예요**
- macOS 아이콘 캐시 문제. Finder/Dock 재시작:
  ```
  killall Dock; killall Finder
  ```
- 또는 Mac 한 번 재부팅

**경고 창이 계속 떠요**
- 매번 뜨면 Gatekeeper 격리 속성이 남아있는 것:
  ```
  xattr -dr com.apple.quarantine /Applications/MeowMode.app
  ```
- 터미널에 위 한 줄 입력하면 끝

**고양이가 화면 끝에서 안 돌아와요**
- 설정 → 활동영역 → "집근처"로 바꾸기 (집 주변 350px 안에서만 돌아다님)
- 또는 "집에 넣기"로 잠시 가두기

**갑자기 멈췄어요**
- 메뉴바 트레이 좌클릭으로 "잠깐 끄기" 상태일 수도 있음. 다시 좌클릭
