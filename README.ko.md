## Meow Mode가 뭐예요?

Meow Mode는 데스크탑 위에서 살아가는 작은 손그림 고양이예요. 화면을 돌아다니고, 자기 집에서 낮잠 자고, 밥그릇에서 먹고, 30가지가 넘는 행동을 해요 — 식빵 굽기, 우다다, 그루밍, 꾹꾹이, 뿌잉뿌잉까지.

- 🐾 **발그림 고양이**가 화면을 돌아다녀요
- 🏠 **자기 집 + 밥그릇/물그릇** — 어디든 끌어다 놓을 수 있어요
- 🎨 **30가지 넘는 행동** (식빵, 우다다, 그루밍, 꾹꾹이, 부비부비, 뿌잉뿌잉…)
- 🔊 **합성 야옹 소리 3종** 중 골라요
- 🌏 **4개 언어**: 한국어 / English / 日本語 / 中文
- 💵 **$2.99**, 평생 무료 업데이트

## 다운로드 & 설치

> **[Gumroad](https://celesteyoon.gumroad.com/l/meowmode)** 에서 받으세요 — $2.99.

1. 내 맥에 맞는 `.dmg` 받기:
   - **애플 실리콘 (M1/M2/M3/M4):** `Meow Mode-x.x.x-arm64.dmg`
   - **인텔:** `Meow Mode-x.x.x.dmg`
   - 모르겠으면? 애플 메뉴  → **이 Mac에 관하여** → 칩 확인.
2. `.dmg` 열고 **Meow Mode**를 **응용 프로그램** 폴더로 끌어다 놓기.
3. **첫 실행:** 앱을 **우클릭 → 열기 → 다시 열기**.
   *(macOS Sequoia: 시스템 설정 → 개인정보 보호 및 보안 → "그래도 열기".)*
   서명 안 한 앱이라 처음 한 번만 필요해요 — 안전합니다.

## 사용법

- 고양이는 알아서 지내요. 끌어서 옮기고, 집도 끌고, 그릇 클릭하면 채워져요.
- **메뉴 막대 아이콘**(우측 상단): 잠깐 끄기, 고양이 찾기, 화면 위치, 종료.
- **설정:** 이름, 활동량, 활동 영역, 소리 종류, 언어 등.
