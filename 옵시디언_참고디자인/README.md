## 🎨 주요 디자인 패턴

- **컬러 팔레트**: 파스텔 핑크, 라벤더, 민트, 피치, 노랑 계열
- **레이아웃**: 카드형 위젯 / 그리드 레이아웃 / 사이드바 네비게이션
- **차트 유형**: 도넛 차트 / 바 차트 / 링 진행률 표시
- **특징**: ADHD 친화적 / 미니멀 / 귀여운 일러스트 아이콘
