## Meow Mode とは？

Meow Mode は、デスクトップの上で暮らす小さな手描きの猫です。画面を歩き回り、おうちでお昼寝し、お皿からごはんを食べ、30種類以上の行動をします — 食パン座り、猫の運動会（ズーミーズ）、毛づくろい、ふみふみ、すねたり…。

- 🐾 **手描きの猫**が画面を歩き回ります
- 🏠 **専用のおうち + ごはん/水のお皿** — 好きな場所にドラッグできます
- 🎨 **30種類以上の行動**（食パン、ズーミーズ、毛づくろい、ふみふみ、すりすり、すね…）
- 🔊 **合成された鳴き声 3種類**から選べます
- 🌏 **4言語**：한국어 / English / 日本語 / 中文
- 💵 **$2.99**、アップデートは生涯無料

## ダウンロードとインストール

> **[Gumroad](https://celesteyoon.gumroad.com/l/meowmode)** で入手できます — $2.99。

1. お使いの Mac に合った `.dmg` をダウンロード：
   - **Apple Silicon（M1/M2/M3/M4）:** `Meow Mode-x.x.x-arm64.dmg`
   - **Intel:** `Meow Mode-x.x.x-x64.dmg`
   - 不明な場合は？ Appleメニュー  → **このMacについて** → チップを確認。
2. `.dmg` を開き、**Meow Mode** を **アプリケーション** フォルダへドラッグ。
3. **初回起動:** アプリを **右クリック → 開く → もう一度「開く」**。
   *(macOS Sequoia: システム設定 → プライバシーとセキュリティ → 「このまま開く」。)*
   公証されていないアプリのため初回のみ必要です — 安全です。

## 使い方

- 猫は自由に過ごします。ドラッグして移動、おうちもドラッグ、お皿をクリックで補充。
- **メニューバーのアイコン**（右上）：一時停止、猫を探す、表示レイヤー、終了。
- **設定:** 名前、活動量、行動エリア、鳴き声の種類、言語など。
