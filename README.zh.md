## Meow Mode 是什么？

Meow Mode 是一只生活在你桌面上的小小手绘猫咪。它会四处走动、在自己的小屋里打盹、从碗里吃东西，还会做 30 多种动作 —— 趴成面包、暴冲（zoomies）、舔毛、踩奶、闹别扭等等。

- 🐾 **涂鸦风格的猫**在屏幕上漫步
- 🏠 **专属小屋 + 食碗/水碗** —— 可拖到任意位置
- 🎨 **30 多种行为**（面包猫、暴冲、舔毛、踩奶、蹭头、闹别扭…）
- 🔊 **3 种合成喵叫声**任你选择
- 🌏 **4 种语言**：한국어 / English / 日本語 / 中文
- 💵 **$2.99**，终身免费更新

## 下载与安装

> 在 **[Gumroad](#)** 获取 *(链接即将上线)*。

1. 下载适合你 Mac 的 `.dmg`：
   - **Apple 芯片（M1/M2/M3/M4）:** `Meow Mode-x.x.x-arm64.dmg`
   - **Intel:** `Meow Mode-x.x.x-x64.dmg`
   - 不确定？点击苹果菜单  → **关于本机** → 查看芯片。
2. 打开 `.dmg`，将 **Meow Mode** 拖入 **应用程序** 文件夹。
3. **首次启动:** 右键点击应用 → **打开** → 再次 **打开**。
   *(macOS Sequoia：系统设置 → 隐私与安全性 → “仍要打开”。)*
   因为应用未经公证，仅首次需要 —— 它是安全的。

## 使用方法

- 猫咪会自己活动。可拖动它、拖动小屋，点击碗就能添食加水。
- **菜单栏图标**（右上角）：暂停、寻找猫咪、显示层级、退出。
- **设置:** 名字、活跃度、活动范围、叫声种类、语言等。
